# HTML Resume Website

This project is a single-page resume website built using **only HTML**.  
The goal of this project is to demonstrate proper HTML structure, semantic elements, and clean layout design based on a provided reference layout.

The resume includes clearly separated sections and uses tables and lists to organize information in a readable and professional format.

---

## Project Overview

This resume website is designed as a simple, clean, and semantic HTML page that closely follows a traditional resume layout. The focus of this project is on:

- Proper use of HTML tags
- Clear layout and structure
- Readable and organized content
- Consistent spacing and alignment
- Semantic HTML5 elements

---

## Features

- Semantic HTML5 structure (`header`, `main`, `section`, `article`, `footer`)
- Resume sections:
  - About Me
  - Skills
  - Experience (Table)
  - Projects
  - Education (Table)
  - Contact Information
- Clean and readable layout
- HTML-only implementation (no CSS or JavaScript)
- Easy to customize content
